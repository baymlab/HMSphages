# Extracting phage genomic DNA

## Objective
To get genomic DNA from phage

## Rationale
Once you have a phage lysate, it is useful to extract phage DNA so that it can be sequenced later on. In this protocol weâ€™re using a kit formulated to extract nucleic acids from viral samples. The process goes like this: The viral particles are first _opened_ using Proteinase K, a Lysis Buffer and heat. Then, ethanol is added and the sample is loaded into a column with a silica matrix. The DNA molecules bind to the silica matrix, and the impurities such as proteins and nucleases are removed by washing them off with the Wash Buffer. Finally the DNA is recovered from the column with molecular grade water, which then we can store and use for sequencing later on.
